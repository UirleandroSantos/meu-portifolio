Criptograma

Este projeto é um criptograma interativo que permite ao usuário transformar qualquer texto digitado em uma versão totalmente codificada, alterando seus caracteres para um formato criptografado e difícil de entender à primeira vista. O usuário pode inserir qualquer mensagem no campo de entrada, e o sistema gera automaticamente o texto criptografado, que pode ser facilmente copiado para ser compartilhado ou armazenado. Além disso, o projeto oferece a funcionalidade reversa: ao colar ou digitar o texto codificado, o usuário pode decodificá-lo instantaneamente para recuperar a mensagem original. Isso torna o criptograma uma ferramenta útil para proteger informações simples, criar mensagens secretas ou apenas se divertir com códigos. A interface é intuitiva e amigável, garantindo que mesmo usuários sem conhecimento técnico possam usar o criptograma com facilidade.
